## 语法

```c++
#include <iostream>

int main() {
    std::cout << "Hello World!" << endl;
    return 0;
}
```

同样

```c++
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    return 0;
}
```
